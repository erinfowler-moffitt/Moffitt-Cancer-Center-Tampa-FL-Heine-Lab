# Automated Breast Density Detection Technique (IDL)

This folder contains IDL code for the Automated Breast Density Detection Technique.

Files included:
- `density_detection.pro` — starter IDL code and function skeleton.
- `.gitkeep` — keeps the folder in git when empty.

Purpose: provide a place for the lab's IDL implementation of breast density detection. Replace the starter code with your implementation and add any datasets, tests, or notes as needed.
